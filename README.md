## 🎯 产品愿景

构建 **"AI 销冠伙伴"** —— 一个部署在企业私域流量池中的 **24/7 自主销售智能体**。

### 核心定位

> 超越传统客服机器人，它是具备 **Top Sales 思维模型** 的主动谈判专家与策略顾问。

### 价值主张

- ✅ **独立闭环处理**：从"破冰"到"成交"的全流程自动化
- ✅ **7x24小时响应**：零时差响应客户需求
- ✅ **智能转人工**：仅在极高风险或复杂情感场景下无缝转接
- ✅ **自我进化能力**：通过历史数据优化话术，根据市场动态调整策略

### 终极形态

> 一个具备 **自我进化能力** 的销售合伙人，成为企业最核心的数字资产。

## 💪 核心能力特征

### 🧠 深度心理与微情绪分析

利用多模态大模型识别文字背后的情绪信号，动态调整沟通策略：

| 情绪类型 | 检测信号 | 响应策略 |
|---------|---------|---------|
| 犹豫 | "我再考虑一下" | 触发SPIN追问痛点 |
| 试探 | "能再优惠吗" | 强调价值而非降价 |
| 急躁 | 连续追问 | 精简回复，快速给结论 |
| 信任 | 正面反馈 | 推进到下一阶段 |

### 📊 动态策略布局 (Dynamic Strategy)

摒弃固定话术树，基于成熟销售方法论实时生成千人千面的对话路径：

- **SPIN 销售**：Situation → Problem → Implication → Need-payoff
- **FAB 销售**：Features → Advantages → Benefits
- **BANT 资格评估**：Budget → Authority → Need → Timeline

### 🔍 事实严谨性 (Grounded RAG)

严格基于企业知识库回答，防止AI幻觉：

```
客户问题 → RAG检索 → 事实校验层 → 准确回复
                    ↓
              不确定时 → 自动转人工 / 承诺核实后回复
```

### 👤 全生命周期画像构建

自动提取并实时更新客户画像：

```
┌─────────────────────────────────────────────────┐
│                   客户画像                        │
├─────────────────────────────────────────────────┤
│ 预算 (Budget)    : ¥50,000 - ¥100,000          │
│ 权限 (Authority) : 技术总监 (决策者)            │
│ 需求 (Need)      : 团队协作、数据分析           │
│ 时间线 (Timeline): Q2 上线                      │
│ 痛点             : 现有工具操作复杂、效率低      │
│ 竞品关注         : 竞品A、竞品B                 │
└─────────────────────────────────────────────────┘
```

## 🚀 快速开始

### 前置要求

- Go 1.21+
- Docker & Docker Compose
- LLM API Key (OpenAI / Anthropic / Azure)

### 一键启动

```bash
# 1. 克隆仓库
git clone <repository-url>
cd salesmate

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env 填入 API Keys

# 3. 启动所有服务（约2分钟就绪）
make docker-up
```

服务启动后：
- **Gateway**: http://localhost:18790
- **Qdrant**: http://localhost:6333
- **Redis**: localhost:6379
- **PostgreSQL**: localhost:5432

### 本地开发

```bash
# 安装依赖
make install

# 初始化知识库
make init

# 运行网关
make run

# 热重载开发 (需要安装air)
make dev

# 运行测试
make test

# 代码检查
make lint
```

### 使用 CLI

```bash
# 启动交互式销售代理
./salesmate agent

# 发送单条消息测试
./salesmate agent -m "你好，我想了解你们的产品"

# 启动网关服务
./salesmate gateway

# 查看系统状态
./salesmate status

# 管理渠道
./salesmate channels status
```

## 🛡️ 安全围栏与人机协作

### 四类安全围栏

| 围栏类型 | 触发条件 | 执行级别 | 响应行为 |
|---------|---------|---------|---------|
| **价格围栏** | 报价低于授权范围 | SoftAlert | 警告但允许继续 |
| **合同围栏** | 做出合同承诺 | HardBlock | 阻止并转人工 |
| **功能围栏** | 声称未验证功能 | Review | 标记需人工复核 |
| **竞品围栏** | 贬低竞品/虚假对比 | Review | 标记需人工复核 |

### 置信度路由

```
                    ┌─────────────┐
                    │  AI 响应生成 │
                    └──────┬──────┘
                           │
                    ┌──────┴──────┐
                    │ 置信度计算   │
                    └──────┬──────┘
                           │
        ┌──────────────────┼──────────────────┐
        │                  │                  │
        ▼                  ▼                  ▼
   > 90%              60% - 90%           < 60%
 高置信度              中置信度            低置信度
        │                  │                  │
        ▼                  ▼                  ▼
  直接自动回复      草稿推送人工确认    立即@人工介入
```

### 情绪熔断机制

```go
// 情绪熔断触发条件
触发场景:
  - 愤怒情绪强度 > 0.7
  - 挫败情绪强度 > 0.7
  - 检测到关键词: "投诉", "领导", "律师", "退款"

触发后动作:
  1. 立即暂停自动回复
  2. 记录情绪历史
  3. 通知人工客服
  4. 发送安抚消息: "我正在为您转接专属客服..."
```
